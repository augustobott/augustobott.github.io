Tomato Firmware Extensions

Just a while ago - some time around 2008 - SDCard support for the WRT54GL ported from OpenWRT. During the next few years, improvements here and there (i.e. 3G USB modem support, Multiple SSID - a.k.a. guest WLAN - PPTP VPN server support and VLAN settings GUI). At last, bandwidth auditing and monitoring (a.k.a. IP Traffic Monitoring). Look for Teaman branches for the code (unmaintained these days - most of the code has been merged into several still active branches).

  • Sources: Github
  • Sources: Original Repository
  • Sources: Main Repository
  • Date: May 2012

Porto Alegre Historical Maps

Using LeaftletJS, these applications allow comparing maps of Porto Alegre and how the city has grown through time - users can select which layers to show, re-order them and play with transparency to see what's beneath a particular place over time.
